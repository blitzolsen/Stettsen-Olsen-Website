UPCLA Membership Database Management Website

The UPLCA is a non-profit organization that is responsible for the CEU's and regulation of the entire industry. They hold annual conferences for the training of the industry. I was commissioned to build a website to manage their membership. This page uses PHP to manage a very small database of less than 500. The UPLCA did not want their members to need passwords or user names. The site is still being build but you may view it by clicking the image to the left or you may wish to view the sorce code here

This is my current project and how I've enjoyed it! Like web design, I love seeing a pretty interface complimented with clean and efficient code. This lacks the interface that I would like, but you can see what it does in the video. My favorite part of what I've made thus far is the fraction class. The class handles the math, simplification, and comparing of all fractions. This keeps the rest of the code neat and tidy. A Singleton pattern is used so that the data can be accessed from all activities without having to pass a bunch of objects. You can see the code for this app in it's Repository
